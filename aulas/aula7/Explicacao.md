## Organização dos dados
- arquivo texto (csv) com palavras ou expressões pejorativas  
- lista para armazenar as palavras ou expressões pejorativas  
- variável texto (digitada pelo usuário)  

## Fluxo ou dinâmica do sistema
1) Executar o sistema em terminal (Command Line Interface)  
2) Carregar as palavras pejorativas do arquivo para dentro da lista  
3) Receber texto do usuário  
4) Analisar se as palavras do texto aparecem na lista de palavras pejorativas 
5) Mostrar as palavras do texto que estão na lista , bem como quantidade delas
